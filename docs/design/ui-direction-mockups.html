Four high-fidelity treatments of the Mappings page. 1 refines the native macOS look; 2–4 are progressively more distinctive. They share the exact same content so only the design language changes.

01 / REFINE

Refined Native

lane a The current System-Settings feel, tightened: leading action icons, soft row cards, crisp keycaps, macOS blue. Feels at home on macOS, just more finished.
02 / DISTINCT

Aurora

lane b A soft gradient-mesh world behind frosted glass. Category-gradient action pills, raised keycaps, a violet→pink brand. Playful-premium, lots of personality.
03 / DISTINCT

Graphite Pro

lane b Dark glass, faint grid, electric-cyan glow. Mono headers, graphite 3D keycaps, neon-edged pills. Reads like a pro developer tool — the keyboard view's vibe, everywhere.
04 / DISTINCT

Editorial Mono

lane a/b Typography-led restraint: a Fraunces serif for titles, monospace nav + keycaps, hairline rows, one vermillion accent, generous whitespace. Calm, magazine-like, unmistakably designed.
